# Модуль предиктивной аналитики и прогнозирования спроса созданный на базе 1С:Розница для казахстана редакция 2.3

Интеллектуальное автономное расширение для прогнозирования продаж и оптимизации управления запасами на предприятиях розничной торговли и общественного питания. Проект успешно защищен на профильной кафедре с оценкой **91 балл (Отлично)**. 

Получена официальная положительная рецензия от ведущего инженера-программиста **ТОО «1С-Рейтинг»** (оценка рецензента — 95/100).

---

## 🚀 Ключевые особенности архитектуры

* **Полная автономность:** Реализовано в виде **расширения конфигурации (`.cfe`)**. Интегрируется в систему без изменения типового кода донора, сохраняя базу полностью обновляемой.
* **Универсальность:** Разработано под базовые метаданные конфигурации `1С:Розница`. Благодаря этому автоматически совместимо с вертикальными отраслевыми решениями (например, `1С:Ресторан`), использующими те же объекты закупкок и продаж.
* **Снижение издержек:** Алгоритм разработан для минимизации дефицита и порчи продукции (потенциальное снижение потерь до 10% за счет исключения человеческого фактора).

---

## 🧠 Математическая модель

В основе расчетного модуля лежит **комбинированный метод прогнозирования**, объединяющий:
1. Расчет средневзвешенных значений по историческим периодам продаж.
2. Факторный анализ (учет внешних событий, маркетинговых активностей, сезонности и праздников через систему весовых коэффициентов).

Оптимизация точности алгоритма производилась путем минимизации среднеквадратичной ошибки (MSE).
